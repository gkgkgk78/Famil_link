### 🎦 주요 기능

1. QR코드를 활용한 간편한 로그인 
2. Teachable Machine을 활용한 안면인식 
3. 보다 정확한 안면인식위한 Mediapipe를 활용한 비디오 데이터 후처리 
4. 캠과 MQTT를 활용한 로그인, 로그아웃, 영상편지 녹화, 전송, 출력
5. 사용자 편의성을 위한 자막과 TTS를 활용한 음성 안내 
6. STT를 활용한 음성 제어
7. 가족 간 연락빈도를 체크 및 안내하여 소통 유도
